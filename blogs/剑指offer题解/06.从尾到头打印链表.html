# 06.从尾到头打印链表

# 方法一：递归法

# 解题思路

可以利用递归，一直走到链表末端，当回溯时依次将节点值加入列表。这样就可以实现倒序输出。

# 算法分析

  • 终止条件：以head == null为终止条件，直接返回。
  • 递推工作：每次传入head.next。
  • 回溯阶段：层层回溯是，将节点值依次加入列表。

# 代码实现

class Solution{
    Arraylist<Integer> tmp = new ArrayList<>();
    public int[] reversePrint(ListNode head){
        recur(head);
        int[] res = new int[tmp.size()];
        for(int i = 0; i < res.length; i++){
            res[i] = tmp.get(i);
        }
        return res;
    }
    
    void recur(ListNode head){
        if(head == null) return; //终止条件
        recur(head.next);		 //开启递归
        tmp.add(head.val);		 //将节点值加入列表 
    }
}

这里分析一下递归。当递归触发终止条件后，回溯时，前面的那些函数才算“执行完毕”，才可以执行下面的tmp.add(head.val)。并且，当返回的时候，返回的是最上层的那个recur函数。（递归要借助栈来存放）而下层的recur函数并没有退出，所以会执行下面的代码。

# 复杂度分析

  • 时间复杂度O(N)：走完链表，递归N次。
  • 空间复杂度O(N)：递归需要O(N)的栈空间。

# 方法二：辅助栈法

# 解题思路

我们可以创建一个辅助栈，然后遍历链表，将节点的值依次入栈。最后再依次出栈用数组返回，就是倒序打印啦。

# 算法步骤

  • 辅助栈：借助LinkedList创建一个辅助栈。
  • 入栈和出栈

# 代码实现

class Solution{
    public int[] reverse(ListNode head){
        LinkedList<Integer> stack = new LinkedList<>();
        while(head != null){
            stack.addLast(head.val);
            head = head.next;
        }
        int[] res = new int[stack.size()];
        for(int i = 0; i < res.length; i++){
            res[i] = stack.removeLast();
        }
    	return res;
    }
}
