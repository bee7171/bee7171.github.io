          17.打印从1到最大的n位数

          bee71 2020-07-01 算法

          # 解题思路

          思考最大的n位数和位数n的关系。最大的n位数记为end。比如n=1，则最大的数为9。以此类推，得关系式：

          end=10n−1end=10^n-1 end=10n−1

          因此定义范围[1,10n−1][1,10^n-1][1,10n−1]和步长1。通过for循环就可以得到结果。

          # 代码实现

          class Solution{
    public int[] printNumbers(){
        int end = (int)Math.pow(10, n) - 1;
        int[] res = new int[end];
        for(int i = 0; i < end; i++){
            res[i] = i+1;            
        }
        retuern res;
    }
}

          # 复杂度分析

          • 时间复杂度O(10n10^n10n)：生成长度为10n10^n10n的数组需要10n10^n10n的时间。
          • 空间复杂度O(1)：建立数组需要常数级额外空间，数组作为返回结果，不计入额外空间的。

          # 大数越界问题

          当n较大时，end会超出int的取值范围。超出范围的数字无法正常存储。
