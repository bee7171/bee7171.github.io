          # 28.对称的二叉树

          # 解题思路

          先要弄懂对称二叉树的定义，如果二叉树中任意两个对称的节点L和R，一定有：

          • L.val和R.val一定是相等的。
          • L.left.val和R.right.val的值一定是相等的。
          • L.right.val和R.left.val的值一定是相等的。

          看到上面的定义，我们就可以从顶至底，依次递归比较每组节点，如果全部堆成，就说明该二叉树是对称的。

          # 算法步骤

          1. 特例处理：先处理特殊情况。如果root为空，则直接返回true。
          2. 返回值：recur(root.left, root.right)。recur是递归函数。
          3. recur函数细节：
            1. 终止条件：
              • 如果L和R同时都越过了叶节点，则该数从顶至底的节点都对称，则说明了该二叉树是对称的。
              • 如果L和R中只有一个越过了叶节点，则说明该二叉树不对称。
              • 如果两个节点的值不相等，直接返回false，说明不对称。
            2. 递推工作：
              • 判断两节点L.left和R.right是否对称。
              • 判断两节点L.right和R.left是否对称。
            3. 返回值：必须要所有对的节点都对称，所以用&&来相连。

          # 代码实现

          class Solution{
    public boolean isSymmetric(TreeNode root){
        //特例处理：如果root为空，则直接返回true
        if(root == null){
            return true;
        }
        return recur(root.left, root.right);
    }
    
    private boolean recur(TreeNode L, TreeNode R){
		if(L == null && R == null){
            return true;
        }
        if(L == null || R == null || L.val != R.val){
            return false;
        }
		return recur(L.left, R.right) && recur(L.right, R.left);
    }
}

          # 复杂度分析

          • 时间复杂度O(N)：每次递归都是一对节点，所以最多执行N/2次。
          • 空间复杂度O(N)：最差情况下，二叉树退化为链表，使用O(N)大小的栈空间。
