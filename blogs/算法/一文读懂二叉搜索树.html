# 一文读懂二叉搜索树

# 什么是二叉搜索树

Binary Search Tree

二叉搜索树也是一棵二叉树，它满足以下特点：

  • 每个节点的值大于其左子树所有节点的值，小于其右子树所有节点的值。
  • 每一棵子树也是一棵二叉搜索树。
  • 该树存储的元素必须具有可比较性。

# 定义一个二叉搜索树

//二叉搜索树存储的元素必须具有可比较性，所以要继承Comparable接口
public class BST<E extends Comparable<E>> {
	//声明一个节点类
    private class Node{
        public E e;
        public Node left;
        public Node right;
        
        public Node(E e){
            this.e = e;
            left = null;
            right = null;
        }
    }
    //首先需要一个根节点
    private Node root;
    //需要一个变量size来记录存储了多少个元素
    private int size;
    
    //创建二叉搜索树的构造函数
    public BST(){
        root = null;
        size = 0;
    }
    
    //返回二叉搜索树中有多少个元素
    public int size(){
		return size;
    }
    
    //返回该二叉搜索树是否为空
    public boolean isEmpty(){
        return size == 0;
    }
}

# 向BST中添加元素

注意点：

  • 如果要添加的元素在BST中已经有这个元素，那么就不做任何处理，就认为树中已经存在了这个元素。
  • 也就是说，二叉搜索树中是不包含重复元素的。

当然二叉搜索树也可以包含重复元素，只需要定义左子树小于等于节点，或者右子树大于等于节点。我们这里学习的二叉搜索树是不包含重复元素的。

//向二叉搜索树中添加新元素
public void add(E e){
    //如果根节点为空，直接添加元素为根节点
    if(root == null){
        root = new Node(e);
        size++;
    }
    else
        add(root, e);
    
}

//向以node为根节点的树添加元素e,递归实现添加新元素
private void add(Node node, E e){
	//终止条件
    if(e.equals(node.e)){
        return;
    }
    else if(e.compareTo(node.e) < 0 && node.left == 0){
		node.left = new Node(e);
        size++;
        return;
    }
    else if(e.compareTo(node.e) > 0 && node.right == 0){
        node.right = new Node(e);
        size++;
        return;
    }
    
	//递推工作
    if(e.compareTo(node.e) < 0){
        add(node.left, e);
    }
    else
        add(node.right, e);
}
